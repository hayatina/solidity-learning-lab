## Solidity Learning Lab

A collaborative smart contract development project completed as part of the Alchemy University – Learn Solidity course.

This repository contains all Solidity exercises, practice contracts, notes, screenshots, and resources completed throughout the course.

## Course Information
        
Course Name: Learn Solidity Syntax
Platform: Alchemy University
Course Focus: Solidity fundamentals, smart contract development, Ethereum interactions, governance systems, inheritance, mappings, structs, arrays, and advanced Solidity concepts.
